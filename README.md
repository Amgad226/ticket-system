# Documentation

Open documentation.html file inside project 

## Prerequisites

- Node.js v16.15.0
- npm 9.6.7

## Getting Started

1. Clone the repository:
  ```shell
       git clone https://github.com/Amgad226/ticket-system.git
  ```
2. run project with nodemon:
  ```shell
       npm run dev 
  ```
 
  3. run project with node:
  ```shell
       npm run start  
  ```
